Welcome to my project built on Angular 17! As a passion project, my primary goal is to explore new concepts in an engaging and enjoyable manner.

In this endeavor, I've utilized a range of technologies to bring our vision to life:
  • Angular 17: Empowering our frontend development with its robust features.
  • TypeScript: Ensuring type safety and enhanced developer experience.
  • HTML & CSS: Crafting the user interface with the building blocks of the web.
  • Java & Spring Boot: Powering our backend API with reliability and scalability.

My frontend is meticulously crafted in Angular, while the backend thrives on the efficiency of Spring Boot. Notably, I've chosen not to integrate a database into this project. Instead, all data is hardcoded, optimizing simplicity and performance, given that my application primarily relies on "get" requests for data retrieval.

Initially, I experimented with hosting my Spring Boot API on render.com. However, due to latency concerns with free hosting, I transitioned to a solution leveraging GitHub for hosting separate JSON files. This decision significantly improved response times, ensuring a smoother experience for the users.
