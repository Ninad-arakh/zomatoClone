This app is build in Angular 17. As a fun project, The main purpose of the application is to learn new things in a fun way.

The technologies used in this project are mentioned below.
  • Angular 17
  • TypeScript
  • HTML
  • Css
  • Java
  • Spring Boot

Front-end is build in Angular. And Back-end is build in spring boot for API. There is no Database is used. So all the data is hard coded. Because it only uses "get" request to get the data from server side.

I used render.com to host the spring boot api. But it was free so the response was very slow to get the data and i only need to get the json so i push the saperate json files on github and it was way faster than render. You can check that out by clicking on following links.
